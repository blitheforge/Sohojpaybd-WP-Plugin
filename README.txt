This plugin allows your customers to pay with Bkash, Nagad, Rocket, and all Bangladeshi payment gateways. Seamless integration with WooCommerce.

== Description ==

SohojpayBD is a simple and efficient payment gateway plugin for Bangladeshi merchants. It supports multiple payment methods, including **Bkash, Nagad, Rocket**, and other local gateways.

### **Features:**
- Seamless **WooCommerce** integration
- Support for **Bkash, Nagad, Rocket**, and other BD gateways
- Secure transactions with SSL
- Easy setup with minimal configuration
- Transaction history and reports

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **WooCommerce > Settings > Payments** and enable **SohojpayBD**.
4. Enter your API credentials in the settings panel.
5. Save changes and start accepting payments!

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =  
Yes, this plugin is designed specifically for WooCommerce. You must have **WooCommerce installed and activated** to use it.

= Is SSL required? =  
Yes, to ensure secure transactions, you must have an **SSL certificate** installed on your website.

= Does it support refunds? =  
Currently, refunds must be processed manually through the payment provider.
